Open Sources

Yet Another Base 62

View the gem

This is the first Rubygem that I wrote with C extension. In fact, this gem can be used to make url-shortener faster by converting integer to base 62. I created this because there are many gems out there which are implemented in pure Ruby. So, I decided to create one with C extension to see how fast it can go. And, the results is amazing.

Backbone Routes

View the plugin

Backbone Routes is to implement Rails like routes based on Backbone. I created this for DailyMuses because my cofounder complained that Backbone doesn't have something like Rails routes. So, I showed him this.

Mongoid Shortener

View the gem

I want a link shortener to work together with Mongoid and YAB62. So, I created this. This in fact is a Rails 3.1 engine that I first created. It is used by DailyMuses.
